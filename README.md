# bpb

[![CI](https://github.com/luanpotter/bpb/actions/workflows/ci.yml/badge.svg)](https://github.com/luanpotter/bpb/actions/workflows/ci.yml)

Trying out Kotlin for CLI while implementing a solution to a puzzle game.

## Setup for vscode

The built-in Kotlin extension for vscode (`fwcd.kotlin`) doesn't support Kotlin
2.3+. Use the JetBrains Kotlin LSP instead:

```bash
./scripts/setup-vscode.sh
```

Then invoke `>Developer: Reload Window`.
